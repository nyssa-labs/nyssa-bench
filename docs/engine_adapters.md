# Engine Adapters

Adapters implement `NyssaEngine`:

```python
class NyssaEngine:
    def load_task(self, task_spec): ...
    def reset(self, seed: int): ...
    def step(self, action): ...
    def render(self): ...
    def get_state(self): ...
    def close(self): ...
```

The API follows Gymnasium's reset and step shape where possible.
